Day 1: Intro to HTML

  • Element Structure:
    <opening_tag> Element content - text or link </closing_tag (if there is one: <br><hr><img> don't have closing tags)>
  • Basic html page structure:
  • <!DOCTYPE html> <== Current standard is HTML 5. DOCTYPE tells browser what to expect. As HTML standards change, this ensures futureproofing. <html> <head> <== contains metadata not displayed in browser; more below <title></title> <== Title appears in browser tab </head> <body> <p>This is where the meat and potatoes lives</p> <== <p> is paragraph tag </body> </html>
  • HTML docs are collection of parent-child relationships
  • <body> <== parent <p>Paragraph</p> <== child </body>
  • Another example: <body> <== parent <div> <== child <h1>Student</h1> <== grandchild of <body>; child of <div>; sibling of next element <p>Get Started</p> <== grandchild of <body>; child of <div>; sibling of prior element </div> </body>

Image tags

  • img tags self closing;
  • have "src" attribute with name:value pairs
  • image location can be web or local
  • <img src="image-location.com" />

Video tags

  • video tags not self closing
  • <video src="myVideo.mp4" width="320" height="240" controls> Video not supported (displayed if vid doesn't display) </video>

Links

External links
  • target attribute describes where pages open
  • "_blank" opens in new window (modern browsers open in new tabs).
Internal links
  • Most sites link to internal
  • relative paths show link locally
    • e.g. ./index.html
  • absolute paths show full link
    • e.g. www.codecademy.com/learn/ruby
  • Use id to direct interal links
  • e.g.:
    • <li><a href="#introduction">Introduction</a></li> <li><a href="#habitat">Habitat</a></li> <li><a href="#media">Media</li></li>
Semantic vs non-semantic tags
  • Semantic tags like <nav> define content inside them
  • Non-Semantic tags like <div> don't
