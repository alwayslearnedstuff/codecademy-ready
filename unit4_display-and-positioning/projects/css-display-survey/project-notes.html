In this project, you will follow step-by-step instructions to fix a fictional survey website. All of the HTML and most of the CSS is intact, however, the CSS display properties have yet to be set.

In order to complete this project, you must know how to set an element's z-index, display value, and position value.

We recommend that you review our CSS Display Lesson before beginning.

The website's existing index.html and style.css files are displayed in the text editor to the right. Good luck!
