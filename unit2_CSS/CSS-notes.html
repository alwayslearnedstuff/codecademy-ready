CSS operators by "selecting" HTML elements and their attributes:

  • CSS uses selectors in the following format with declaration containing a property:value; relationship:
  • p { color: blue; }
  • selectors can access not just HTML tags (e.g. p, h1, div) but also attributes: e.g. (class, id)
  • Here's an HTML element with attribute:

    and it's selector:

  • .class { color: gray; }
Specificity
  • CSS order of operations by first to last: id, class, tag
  • CSS chaining
  • You can "chain" elements and classes, e.g.: h1.special
  • Selecting child elements
  • This selects the h5 element in div with class description:
  • div.description h5 {color: blue;}
  • The p elements in the main class below will be red, while rest in doc will be blue:
  • p { color: blue; } .main p { color: red; }
  • You can avoid redundancy by applying same styles to multiple different selectors, e.g.:
  • h3, p { color: blue; }
CSS Visual Rules
  • font type: font-family: Garamond;
  • font size: font-size: 10px;
  • font weight, e.g. bold: font-weight: bold
    • You can have all font bold, then one paragraph not, by doing this:
    • font-weight: normal;
  • text auto aligns left inside elements. To change that: text-align: right;
  • Two kinds of color: foreground and background
    • foreground deals with element text
    • background deals with the element's background
      • , e.g.: h1 { color: red; background-color: blue; }
  • opacity controls transparency where 0 is transparent and 1 is opaque, e.g.: .overlay { opacity: 0.5; }
  • CSS can change the background of an element such as with an image, e.g.
    • external: .main-banner { background-image: url("https://www.example.com/image.jpg"); }
    • local: .main-banner { background-image: url("images/mountains.jpg"); }
